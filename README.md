# Индустриальная разработка ПО

Репозиторий содержит выполненные задания по курсу **"Индустриальная разработка ПО"**.

**Структура репозитория**

- **java/** - Программирование на языках высокого уровня

  - `task1/` - Практика 1: Консольное приложение для работы банка
    
- **system_design/** - Проектирование и архитектура программных систем
  
  - `sem1/` - Семинар 1: Моделирование спутниковой группировки
  
  - `sem2/` - Семинар 2: Реализация принципов SOLID
  
  - `sem3/` - Семинар 3: Внедрение DI и Spring Boot
    
  - `task1/` - Практика 1: Закрепление принципов SOLID. Зоопарк
